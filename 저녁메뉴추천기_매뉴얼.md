# 🍽️ 저녁 메뉴 추천기 사용 매뉴얼

## 1. 프로그램 개요
이 프로그램은 매일 저녁 어떤 메뉴를 먹을지 고민하는 분들을 위해 만들어진 **'저녁 메뉴 추천'** 데스크톱 앱입니다.
한식, 중식, 일식, 양식 등 원하는 카테고리를 선택하면 해당 카테고리 내에서 무작위로 맛있는 메뉴를 하나 추천해 줍니다. 만약 특정 종류가 끌리지 않는다면 "아무거나 다 좋아요!" 버튼을 통해 전체 메뉴 중 무작위로 하나를 추천받을 수도 있습니다.

## 2. 사용 방법
1. `.exe` 실행 파일을 더블클릭하여 프로그램을 실행하면 **"저녁 메뉴 추천기"** 창이 나타납니다.
2. 화면 중앙에 있는 메뉴 종류 버튼 중 하나를 클릭해 보세요.
   * **🍚 한식**: 김치찌개, 된장찌개, 삼겹살, 제육볶음 등 다양한 한식 메뉴 중 하나를 추천합니다.
   * **🍜 중식**: 짜장면, 짬뽕, 마라탕, 탕수육 등 인기 중식 메뉴 중 하나를 추천합니다.
   * **🍣 일식**: 초밥, 돈까스, 라멘, 소바 등 일식 메뉴 중 하나를 추천합니다.
   * **🍕 양식**: 피자, 파스타, 스테이크, 리조또 등 양식 메뉴 중 하나를 추천합니다.
3. 딱히 선호하는 종류가 없다면 하단의 **"🎲 아무거나 다 좋아요!"** 버튼을 클릭하세요. 전체 메뉴가 포함된 목록에서 무작위로 하나를 추천해 줍니다.
4. 버튼을 누르면 즉시 하단에 **"오늘의 추천 메뉴는 ✨ [추천 메뉴] ✨ 어떠신가요?"**라는 문구와 함께 결과가 표시됩니다.
5. 다른 메뉴를 추천받고 싶다면, 버튼을 계속해서 눌러 새로운 메뉴를 추천받을 수 있습니다.

## 3. 프로그램 종료
* 더 이상 추천이 필요 없으시다면 창의 우측 상단에 있는 **'X'** 버튼을 클릭하여 프로그램을 종료합니다.

## 4. 참고 사항
* 본 프로그램은 인터넷 연결 없이도 오프라인 환경에서 자유롭게 동작합니다.
* 사용자의 취향에 따라 파이썬 소스 코드(`menu_recommender.py`) 내의 `menus` 딕셔너리 내용을 수정하여 직접 원하는 메뉴를 추가하거나 삭제할 수 있습니다.
